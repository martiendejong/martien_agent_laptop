## Value Distillation Questions

Use these in Phase 2 to identify core value:

1. **Job-to-be-Done:** What job is the user hiring this product to do?
2. **Alternatives:** What do users do today without this product?
3. **Switching Cost:** What would make a user switch from competitors?
4. **Retention Test:** What keeps users coming back?
5. **Referral Test:** What makes users tell others?
6. **Emotional Benefit:** How does this make users feel?
7. **Functional Benefit:** What can users accomplish?
8. **Social Benefit:** How does this affect status/relationships?
9. **Transformation:** What changes in the user's life?
10. **Irreplaceable:** What would users lose if this disappeared?

## Billion-Dollar Feature Tests

Apply these tests in Phase 4:

### The "Killer Feature" Test
- Would users switch from competitors for this alone?
- Is this feature worth the entire product price?
- Would this feature get media coverage?

### The "Unique Value" Test
- Can this value be obtained elsewhere?
- How easily can competitors copy this?
- What makes this defensible?

### The "Network Effects" Test
- Does this get better with more users?
- Do users create value for each other?
- Is there lock-in (ethical)?

### The "Discovery" Test
- Will users discover this on their own?
- Is the value immediately obvious?
- Does this create "wow" moments?

### The "Habit Formation" Test
- Does this create a daily habit?
- Is there a reason to return?
- Does behavior compound?

### The "Viral Coefficient" Test
- Do users naturally share this?
- Is there social proof?
- Is there incentive to invite others?

### The "Monetization" Test
- Will users pay for this?
- Does this expand use cases?
- Does this increase willingness to pay?

## ROI Calculation Frameworks

Use these in Phase 6:

### Value Score Components

**User Impact (0-30 points):**
- Time saved per user per day
- Frustration eliminated
- Capability unlocked
- Delight created

**Business Impact (0-30 points):**
- Revenue increase (direct)
- Cost reduction
- Conversion rate lift
- Retention improvement

**Competitive Advantage (0-20 points):**
- Unique in market
- Difficult to replicate
- Creates switching cost
- Enhances positioning

**Network Effects (0-10 points):**
- Gets better with scale
- Users help each other
- Creates lock-in
- Viral potential

**Strategic Alignment (0-10 points):**
- Aligns with vision
- Enables future features
- Builds capabilities
- Strengthens brand

### Effort Score Components

**Development Time (0-30 points):**
- 0-10: < 1 week
- 11-20: 1-4 weeks
- 21-30: > 1 month

**Technical Complexity (0-25 points):**
- 0-5: Simple CRUD
- 6-10: Standard features
- 11-15: Complex logic
- 16-20: Novel algorithms
- 21-25: Research required

**Dependencies (0-15 points):**
- 0-5: Self-contained
- 6-10: Internal dependencies
- 11-15: External dependencies

**Risk (0-15 points):**
- 0-5: Low risk
- 6-10: Medium risk
- 11-15: High risk (tech/market)

**Team Capacity (0-15 points):**
- 0-5: Team has expertise
- 6-10: Need some learning
- 11-15: Need new hires/skills

## Example Expert Panel Session

### Scenario: Analyzing a To-Do App

**Question:** What's the core value of this to-do app?

**Software Architect:**
"The value is in reliable state sync across devices. Users trust their tasks are always available."

**Behavioral Psychologist:**
"The value is in reducing anxiety about forgetting things. It's an external memory that reduces cognitive load."

**Product Manager:**
"The value is in simplicity. Unlike competitors, this just works without complex GTD methodology."

**Power User:**
"The value is in keyboard shortcuts. I can capture thoughts faster than writing on paper."

**Casual User:**
"The value is that I don't have to think about it. I just add tasks and it reminds me."

**Consensus:**
"Core value is peace of mind through frictionless capture and reliable recall."

**Value Amplification Ideas:**
- What if it captured tasks from voice/email automatically? (reduces friction)
- What if it predicted what you'll forget? (enhances reliability)
- What if it suggested optimal times to do tasks? (enhances peace of mind)
