### Evaluation Criteria

**Value (1-10):**
- Time saved
- Error prevention
- Quality improvement
- Team impact
- Business value

**Effort (1-10):**
- Implementation complexity
- Maintenance burden
- Learning curve
- Dependencies
- Testing requirements

**Ratio Calculation:**
```
Ratio = Value / Effort
```

**Tier Classification:**
- **S+ (Ratio > 8.0):** Exceptional value, minimal effort
- **S (Ratio 6.0-8.0):** High value, low effort
- **A (Ratio 4.0-6.0):** Good value, moderate effort
- **B (Ratio 2.0-4.0):** Moderate value, moderate effort
- **C (Ratio < 2.0):** Lower value or higher effort
