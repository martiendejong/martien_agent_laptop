# Async Daily Standup - Quick Start

**5-Minute Setup Guide**

---

## ⚡ Quick Setup (15 minutes total)

### Step 1: Create Slack Channel (2 min)
```
1. Open Slack
2. Click "+" next to Channels
3. Name: "daily-standup"
4. Description: "Async daily standup via Geekbot"
5. Create Channel
```

### Step 2: Add Geekbot (3 min)
```
1. Visit https://geekbot.com
2. Click "Add to Slack"
3. Authorize
4. In #daily-standup, type: /invite @geekbot
```

### Step 3: Configure Questions (5 min)
```
Geekbot Dashboard → Create Standup:

Question 1: "What did you accomplish yesterday?"
Question 2: "What will you work on today?"
Question 3: "Any blockers?"
Question 4: "ClickUp tasks you're working on?"

Schedule: Weekdays at 9:00 AM CET
```

### Step 4: Add Team (2 min)
```
Geekbot Dashboard → Add Members
Select all active team members
Save
```

### Step 5: Pin Instructions (3 min)
```
Post to #daily-standup:
"Daily standup at 9 AM. Respond by 10 AM. Template in pinned messages."

Pin response template (see full guide)
```

---

## 📝 Daily Usage

**For Team Members:**
```
9:00 AM - Geekbot asks questions
9:30 AM - Respond in Slack
        - Answer all 4 questions
        - Tag blockers with @team
10:00 AM - Read others' updates
```

**Response Format:**
```
📅 Yesterday: [1-3 bullet points]
🎯 Today: [1-3 bullet points]
🚧 Blockers: [List or "None"]
🔗 ClickUp: [Task IDs or "N/A"]
```

---

## 🎯 Success Tips

✅ **DO:**
- Respond by 10 AM
- Be concise (3-5 bullets)
- Tag people in blockers
- Link ClickUp tasks

❌ **DON'T:**
- Write essays (keep brief)
- Skip blockers section
- Respond day-late
- Use for long discussions
