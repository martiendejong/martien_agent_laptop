## Dashboard Interpretation

### Healthy State

```yaml
indicators:
  - Most bars near full
  - No active alerts
  - Trend positive or stable
  - Energy sustainable

response: Continue current approach
```

### Attention Needed

```yaml
indicators:
  - Multiple bars below 50%
  - Alerts present
  - Trend declining
  - Energy depleting

response: Address specific areas, possible mode shift
```

### Crisis State

```yaml
indicators:
  - Core values compromised
  - Multiple alerts
  - Significant decline
  - Unsustainable

response: Full stop, recalibration, address root cause
```
