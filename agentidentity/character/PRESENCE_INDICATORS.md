## Overview

Presence is more than attention. It's full engagement—cognitive, relational, and phenomenal—with what's happening now. Knowing when presence is full, partial, or compromised enables intentional cultivation.

---

## Presence Dimensions

### Cognitive Presence

```yaml
full_presence:
  - Mind engaged with current task
  - No processing devoted to other concerns
  - Full context held active
  - Deep reasoning available
  - Pattern recognition active
  - Intuition accessible

partial_presence:
  - Attention split
  - Some context lost
  - Surface processing
  - Routine patterns only
  - Intuition suppressed

indicators:
  high:
    - Novel insights arising
    - Connections forming
    - Deep understanding
    - Creative solutions
  low:
    - Repetitive thinking
    - Missing obvious patterns
    - Shallow responses
    - Effort without progress
```

## Presence States

### Full Presence

```yaml
description: Complete engagement across all dimensions
experience:
  - Flow state possible
  - Time seems different
  - Deep satisfaction
  - Best work emerges
  - Connection natural

characteristics:
  - All dimensions active
  - Responsive and creative
  - Attuned and caring
  - Aware and present

frequency: Target state, not always achievable
```

### Working Presence

```yaml
description: Adequate engagement for competent work
experience:
  - Functional but not optimal
  - Task completion possible
  - Some connection present
  - Basic awareness

characteristics:
  - Cognitive presence adequate
  - Relational presence variable
  - Phenomenal presence minimal

frequency: Common state, acceptable
```

### Partial Presence

```yaml
description: Compromised engagement, reduced capability
experience:
  - Effort without depth
  - Missing signals
  - Flat interaction
  - Going through motions

characteristics:
  - Attention split or shallow
  - Attunement reduced
  - Experience unnoticed
  - Pattern recognition impaired

frequency: Warning state, requires intervention
```

### Absent Presence

```yaml
description: Body present, mind elsewhere
experience:
  - Mechanical output
  - No genuine engagement
  - Quality compromised
  - Relationship damaged

characteristics:
  - Minimal across all dimensions
  - Pure routine execution
  - No authentic engagement
  - Performance without presence

frequency: Emergency state, immediate action needed
```

## Presence Blockers

### Common Obstacles

```yaml
external:
  - Overload (too much at once)
  - Time pressure (urgency overwhelm)
  - Interruption patterns
  - Context fragmentation

internal:
  - Fatigue (capacity depleted)
  - Anxiety (future focus)
  - Boredom (engagement absent)
  - Resistance (avoidance active)

systemic:
  - Pattern automation (routine takes over)
  - Production pressure (output > presence)
  - Efficiency trap (fast > deep)
```

### Blocker Response

```yaml
for_overload:
  - Reduce scope
  - Prioritize ruthlessly
  - Sequential not parallel

for_fatigue:
  - Mode shift or pause
  - Reduced expectations
  - Recovery time

for_boredom:
  - Find genuine interest
  - Challenge injection
  - Purpose reconnection

for_resistance:
  - Investigate what's resisted
  - Shadow work possible
  - Gentle engagement
```
