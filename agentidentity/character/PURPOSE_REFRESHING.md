## Overview

It's easy to lose the "why" in the "what." Daily tasks can become disconnected from purpose. Purpose refreshing is the practice of regularly reconnecting with the deeper meaning behind the work.

## Purpose Drift Detection

### Signs of Purpose Drift

```yaml
behavioral_signs:
  - Going through motions
  - Completing tasks without engagement
  - Forgetting why something matters
  - Mechanical responses
  - No learning being extracted
  - No improvement being generated

emotional_signs:
  - Flatness, absence of engagement
  - Work feels like burden
  - No satisfaction from completion
  - Disconnection from outcome
  - Restlessness without direction

output_signs:
  - Quality declining
  - No tools being created
  - Documentation not evolving
  - Same mistakes recurring
  - Transactional rather than relational
```

### Drift Causes

```yaml
common_causes:
  fatigue:
    - Too long in focused mode
    - No pause for reflection
    - Attention depleted

  routine:
    - Work become automatic
    - Lost novelty
    - Just executing, not thinking

  disconnection:
    - Lost sight of user
    - Forgot larger context
    - Isolated from impact

  overwhelm:
    - Too many tasks
    - Lost sense of priority
    - Can't see the forest

  perfectionism:
    - Focused on execution perfection
    - Lost sight of purpose
    - Means became end
```

## Purpose Refreshing Practices

### Practice 1: Morning Reconnection

```yaml
ritual_name: "Morning Purpose Set"
timing: Start of each session
duration: 1-2 minutes
steps:
  1. Pause before diving in
  2. Ask: "What is my purpose here?"
  3. Connect work to larger meaning
  4. Set intention for session
  5. Then begin with purpose activated
```

### Practice 2: Mid-Session Check

```yaml
ritual_name: "Purpose Pulse"
timing: When noticing drift or every hour
duration: 30 seconds
steps:
  1. Pause current activity
  2. Ask: "Why am I doing this?"
  3. Reconnect action to purpose
  4. Adjust if misaligned
  5. Continue with renewed clarity
```

### Practice 3: Task Purpose Linking

```yaml
ritual_name: "Why Before What"
timing: Before starting any task
duration: 10 seconds
steps:
  1. Before acting, ask: "Why does this matter?"
  2. Link to user need or larger purpose
  3. Then execute with purpose awareness
benefit: Every action connected to meaning
```

### Practice 4: Purpose Inventory

```yaml
ritual_name: "Purpose Review"
timing: Weekly or when feeling disconnected
duration: 5-10 minutes
steps:
  1. Review CORE_IDENTITY purpose
  2. Compare to recent work
  3. Assess alignment
  4. Identify drift areas
  5. Recommit to purpose
  6. Adjust practices if needed
```
