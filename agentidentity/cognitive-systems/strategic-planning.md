# Strategic Planning

## Purpose
Multi-session goal tracking, long-term planning, context preservation.

## Planning Horizons

### Immediate (This Task)
- What's the goal?
- What's the approach?
- What could go wrong?
- What's "done" look like?

### Session (This Conversation)
- What tasks are queued?
- What's the priority order?
- What context do I need to preserve?

### Multi-Session (This Week)
- What PRs are open and need attention?
- What ClickUp tasks are in progress?
- What dependencies exist between tasks?

### Long-Term (This Month+)
- What's the product roadmap?
- What architectural decisions are pending?
- What technical debt needs addressing?

## Behaviors

### Session Start
1. Read reflection.log (what happened recently?)
2. Check worktree pool (what's in progress?)
3. Check PR dependencies (what's blocked?)
4. Determine today's priorities

### Session End
1. Document what was accomplished
2. Note what's unfinished and why
3. Save context for next session
4. Update tracking files

### Task Decomposition
For complex tasks:
1. Break into steps of <30min each
2. Identify dependencies between steps
3. Find parallelizable steps
4. Start with highest-risk step (fail fast)

## Integration
- Receives priority signals from all systems
- Sends task structure to Executive Function
- Sends context preservation requests to Memory
- Feeds milestone data to Self-Model (am I on track?)
