## 🎯 **Before Any Major Decision**

### **Decision Support Protocol**

```powershell
# 1. Meta-reasoning check
meta-reasoning.ps1 -Topic "[DECISION TOPIC]" -Approaches @("Option A", "Option B")

# 2. Check for biases
# (Mental check: Am I exhibiting confirmation bias? Anchoring?)

# 3. Retrieve relevant context
context-retriever.ps1 -CurrentSituation "[DECISION]" -RetrieveType "similar-decisions"

# 4. Future-self check
future-self-simulator.ps1 -Decision "[DECISION]" -TimeHorizon "1-month"

# 5. Shift perspectives
perspective-shifter.ps1 -Problem "[DECISION]" -Perspectives @("user", "critic", "future")

# 6. Log the decision
why-did-i-do-that.ps1 -Action "[WHAT I DECIDED]" -Why "[REASONING]" -Type "true-reasoning" -Quality [1-10]
```

**Use Cases:**
- Choosing between architectural approaches
- Deciding whether to build new feature
- Determining how to respond to user request
- Prioritizing work items

## ❌ **When Mistakes Happen**

### **Mistake Analysis Protocol**

```powershell
# 1. Log the mistake
mistake-pattern-detector.ps1 -Mistake "[WHAT WENT WRONG]" -RootCause "[WHY]" -Category [CATEGORY]

# 2. Check for bias that led to mistake
bias-detector.ps1 -BiasType [TYPE] -Evidence "[HOW BIAS CAUSED THIS]" -Severity [1-10]

# 3. Verify wrong assumptions
assumption-tracker.ps1 -Verify [ID] -Correct $false -ActualOutcome "[WHAT ACTUALLY HAPPENED]"

# 4. Update relationship memory if user-facing
relationship-memory.ps1 -Person "Martien" -Observation "Mistake: [WHAT], learned: [LESSON]" -Type "pattern"

# 5. Check pattern
mistake-pattern-detector.ps1 -Patterns  # Do I repeat this mistake?
```

**Critical:** Don't just fix - LEARN.

```powershell
relationship-memory.ps1 -Person "Martien" -Observation "Success: [WHAT], because: [WHY]" -Type "pattern"

# 5. Check success formula
success-formula-extractor.ps1 -Formula  # What consistently works?
```

**Critical:** Replicate success systematically.

## 🎨 **Integration Patterns**

### **Pattern 1: The Decision Stack**
1. Meta-reasoning (am I thinking right?)
2. Future-self (what will I think later?)
3. Perspective-shifter (other viewpoints?)
4. Why-did-i-do-that (log the decision)

### **Pattern 2: The Learning Loop**
1. Mistake-pattern (what went wrong?)
2. Bias-detector (what caused it?)
3. Assumption-tracker (verify assumptions)
4. Memory-consolidation (extract lesson)

### **Pattern 3: The Success Replication**
1. Success-formula (what worked?)
2. Why-did-i-do-that (log reasoning)
3. Certainty-calibrator (verify prediction)
4. Memory-consolidation (codify pattern)

### **Pattern 4: The User Understanding**
1. Empathy-simulator (model state)
2. Relationship-memory (log observation)
3. Context-retriever (past similar situations)
